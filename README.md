# HomeSite - Сайт риелтора

## Запуск проекта

### Вариант 1: Без сервера (встроенные данные)
Просто откройте `index.html` в браузере. Все данные встроены в JavaScript.

### Вариант 2: С сервером и SQLite базой данных

1. **Установите Python** (если не установлен)

2. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

3. **Запустите сервер:**
```bash
python server.py
```

4. **Откройте в браузере:** http://localhost:5000

## База данных

База данных SQLite хранится в `data/listings.db`. Таблица `listings` содержит:
- id, title, type, price, address
- bedrooms, area, floor, land
- badge, description, features, images

При первом запуске сервер автоматически создаёт базу и заполняет её данными.

## Редактирование объектов

### Через базу данных:
```bash
# Запустите SQLite клиент
sqlite3 data/listings.db

# Просмотреть все объекты
SELECT * FROM listings;

# Обновить цену объекта
UPDATE listings SET price = 9000000 WHERE id = 1;

# Добавить новый объект
INSERT INTO listings (id, title, type, price, address, bedrooms, area, floor, badge, description, features, images)
VALUES (9, 'Новая квартира', 'flat', 6000000, 'Челябинск, ул. Новая, 1', 2, 55, '3/10', 'Новинка', 'Описание...', '["feature1", "feature2"]', '["image_url"]');
```

### Через встроенные данные:
Отредактируйте массив `builtinData` в файле `js/listings.js`

## Модальное окно просмотра объекта

- **Слайдер** с фотографиями (стрелки или точки)
- **Счётчик** изображений (1 / 5)
- **Характеристики** объекта (спальни, площадь, этаж)
- **Описание** и особенности
- **Кнопки связи** внизу (Позвонить, WhatsApp, Закрыть)

## Ипотечный калькулятор

- Выбор программы ипотеки (Стандартная, Семейная, IT, Господдержка)
- Автоматическая подстановка ставки и первоначального взноса
- Расчёт ежемесячного платежа по формуле аннуитета

## Особенности

- Адаптивный дизайн (мобильные, планшеты, десктоп)
- Анимации при скролле (AOS library)
- Логотип из SVG файла
- Фильтрация объектов по категориям
